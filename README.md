# Sudoku-Solver
Solves sudoku puzzles using DFS backtracking algorithm. Programmed in Racket.

Run program by downloading sudoku-project.rkt and inside file, call function (solve-latin sudoku? insert-puzzle). Place a list of 9 strings (rows), each with 9 characters (empty boxes should be represented by question marks in each string). Run sudoku-project.rkt to get answer.
